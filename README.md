# pokemonMiniGame

一个 web 端的宝可梦小游戏

- author：肖楷炫
- authorEmail：xiaokaixuan24@163.com

## frontend 目录

- web 端源码

```bash
cd frontend

npm i

npm run dev
```

## server 目录

- 服务端源码

```bash
cd server

npm i

npm start
```

### 注意

- 需要本地启动 MongoDB
- 数据库配置在\server\config/db.config.js 中改
- 宝可梦数据初始化 get 请求一下/pokemons/init 就行
